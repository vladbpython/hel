## [1.1.1]

### Fixed

- **`SeqInner` / `SingleInner`**: slot buffer is now heap allocated
  (`Box<[Slot<T>; CAP]>`) instead of living inline in the struct. With large
  `T` or large `CAP` the inline array could overflow the thread stack when a
  channel was created; moving it to the heap fixes this. The hot path is
  unaffected slots stay in one contiguous block and `T` remains inline.

- **`SingleInner::yield_before_park`**: now returns `false` only on macOS
  other platforms use their own value. 
  This tunes the spin/yield-before-park behavior per scheduler.

- **`internal_core/errors.rs`**: error types no longer require `T: Debug`.
